# Glossary

- **Asset**: A binary resource like an image or a game file.
- **Async Trait**: A Rust trait that supports asynchronous methods (via the `async-trait` crate).
- **Brute Force**: In the context of this launcher, a deep and thorough scan of every directory on a disk.
- **Crate**: The basic unit of code organization in Rust.
- **GDK**: GTK's drawing kit; used for handling windowing and drawing.
- **GTK4**: The fourth major version of the Gnome ToolKit, used for building the GUI.
- **Hexagonal Architecture**: A design pattern that uses "Ports and Adapters" to isolate the application core.
- **MOD**: A file extension used for game modifications in the Gothic series.
- **OpenGothic**: A modern re-implementation of the Gothic engine.
- **Port**: An interface defining a boundary between layers.
- **VDF**: Virtual Disk File; the legacy archive format of the ZenGin engine.
- **Workspace**: A set of Rust packages that share a common `Cargo.lock` and output directory.
